# 音乐专辑项目分析展示

一个用于分析和展示音乐专辑数据的项目。

## 项目功能

- 音乐专辑数据导入
- 数据统计分析
- 可视化展示

## 技术栈

- Python
- NumPy / Pandas
- Matplotlib / Seaborn

## 目录结构

```
music-album-analysis/
├── data/           # 数据文件
├── src/            # 源代码
├── notebooks/      # Jupyter笔记本
├── README.md
└── .gitignore
```

## 使用方法

```bash
# 安装依赖
pip install -r requirements.txt

# 运行分析
python src/main.py
```
